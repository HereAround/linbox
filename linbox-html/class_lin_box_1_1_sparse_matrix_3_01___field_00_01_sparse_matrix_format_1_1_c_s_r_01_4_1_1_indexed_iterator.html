Detailed Description

template<class _Field>
class LinBox::SparseMatrix< _Field, SparseMatrixFormat::CSR >::IndexedIterator

a forward iterator.

Allows traversing the nonzero values, with position info. Used, for example, in building an equivalent dense matrix.

Preincrement, ++i, is provided, but not postincrement, i++. Dereference, *i, is replaced by i.rowIndex(), i.colIndex(), i.value().

No distinction is made between IndexedIterator and ConstIndexedIterator. In particular, value() is not a reference. To modify the matrix use setEntry().
