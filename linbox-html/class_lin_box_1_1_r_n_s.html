Detailed Description

template<bool Unsigned>
class LinBox::RNS< Unsigned >

RNS.

Creates a RNS than can recover any number between 0 and q-1 if Unsigned=true or -q+1 and q-1 otherwise (where q=2 <up>bits </up>).
