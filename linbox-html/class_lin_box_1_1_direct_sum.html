Detailed Description

template<class _Blackbox1, class _Blackbox2>
class LinBox::DirectSum< _Blackbox1, _Blackbox2 >

If C = DirectSum(A, B) and y = xA and z = wB, then (y,z) = (x,w)C.

And similarly for apply.
