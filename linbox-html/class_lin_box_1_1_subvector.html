Detailed Description

template<typename Iterator, typename ConstIterator = Iterator>
class LinBox::Subvector< Iterator, ConstIterator >

Dense subvector

This class provides a statically sized subvector of a random access container (such as std::vector, deque).

It does not work on sparse linbox vectors. It implements all of the types and methods of a std::vector except for those that invalidate iterators, i.e., those (potentially) involving vector resizing, such as push_back(), insert(), resize().
